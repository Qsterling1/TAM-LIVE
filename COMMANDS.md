# Commands (quick reference)
- `npm install` – install project dependencies.
- `npm start` – run the web console locally.
- `npm run rag:index -- --input "Q:\\FakeDesktop\\file\\create\\FlyAndShadow_3D"` – rebuild the local RAG embeddings cache from your Unity folder.
- `npm run build` – create a production build in `build/`.
- `set REACT_APP_AI_ENGINE=lmstudio` (or `google|openai|anthropic`) – pick which AI backend to use at runtime.
